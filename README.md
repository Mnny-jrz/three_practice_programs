# Activity 3: Programming Challenges

This repository contains practice programs for Activity 3 in our programming course.  
Each file demonstrates basic Python concepts such as input handling, conditionals, and arithmetic.

## Files Included
- **bigger_number.py** → asks for two numbers and prints the bigger one
- **equal_numbers.py** → checks if two numbers are equal
- **sum_two_numbers.py** → takes two numbers and prints their sum
